# GrafoPlanoCartesiano
Projeto de estudo em Programação Orientada a Objetos (POO) usando Java, que simula um plano cartesiano onde pontos nomeados podem ser criados, manipulados e conectados automaticamente formando um grafo de rotas.  O sistema permite realizar operações geométricas básicas e também calcular o menor caminho entre dois pontos com o algoritmo de Dijkstra
